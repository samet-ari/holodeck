# Guide d'Installation Holodeck

## Prérequis
- VMware Workstation/Player
- Debian 12 ISO
- 8GB RAM minimum

## Installation des VMs
1. Créer VM Serveur (sans GUI)
2. Créer VM Client (avec GNOME)
3. Configurer réseau Host-only

## Services à installer
- Nginx + SSL
- PHP 7.4 + 8.2
- MariaDB
- DNS/DHCP
- FTP sécurisé
- LDAP
